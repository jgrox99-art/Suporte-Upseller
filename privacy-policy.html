Política de Privacidade

Extensão: UpSeller - Respostas Rápidas
Última atualização: 14 de abril de 2026

1. Informações que coletamos

A extensão "UpSeller - Respostas Rápidas" não coleta, transmite ou armazena nenhum dado pessoal do usuário em servidores externos.

Todos os dados são armazenados localmente no navegador do usuário por meio da API chrome.storage.local, incluindo:

  • Posição vertical do widget na tela
  • Estado do widget (aberto, minimizado ou fechado)
  • Lista de artigos favoritados pelo usuário (apenas URLs)

2. Dados embutidos na extensão

A extensão contém dados estáticos pré-carregados que não dependem de coleta:

  • 112 respostas rápidas para atendimento ao cliente (texto fixo)
  • 824 artigos indexados da Central de Ajuda da UpSeller (títulos e links públicos)

Esses dados são públicos e disponíveis no site upseller.com/pt/help.

3. Permissões utilizadas

  • storage: Para salvar preferências locais do usuário (posição do widget, favoritos).
  • activeTab: Para copiar texto para a área de transferência quando o usuário seleciona uma resposta rápida.
  • clipboardWrite: Para permitir a cópia de respostas e links para a área de transferência.

4. Comunicação com servidores externos

A extensão não faz nenhuma requisição a servidores externos. Toda a busca é realizada localmente nos dados embutidos. Não há analytics, rastreamento ou telemetria.

5. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos nenhum dado do usuário a terceiros.

6. Segurança

Os dados armazenados localmente são protegidos pelos mecanismos de segurança do próprio Google Chrome e do sistema operacional do usuário.

7. Alterações nesta política

Quaisquer alterações nesta política de privacidade serão publicadas nesta página e refletidas na data de última atualização acima.
